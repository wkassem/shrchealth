# shrchealth

Professional one-page website for **SHRC – Saad Halimi Rehabilitation Center**.

## Run locally

Open `index.html` directly in a browser, or serve the folder with a static server.

## Variant URLs

- `/` → active home page (currently renders `App.jsx`)
- `/app` → renders `App.jsx`
- `/app1` → renders `App1.jsx`
